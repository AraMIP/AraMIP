# [Arabic Metaphor Identification Procedure (AraMIP)](https://github.com/AraMIP/AraMIP/)

&nbsp;
## ℹ️ About
The Arabic Metaphor Identification Procedure (AraMIP) is a two-stage annotation framework for identifying metaphor in Arabic texts. It adapts the Metaphor Identification Procedure Vrije Universiteit (MIPVU) to Arabic while incorporating categories from the Arabic rhetorical tradition (*al-balāgha*).

This repository provides the official annotation guidelines and related project resources.

&nbsp;
### 🎯 Purpose
The purpose of this repository is to:

* provide the official AraMIP annotation guidelines
* support consistent metaphor annotation across Arabic corpora
* facilitate reproducible research in Arabic figurative language processing.

&nbsp;
### 📦 Repository Contents
* AraMIP_Annotation_Instructions.pdf — Official annotation guidelines for the AraMIP framework.
* static/ — Repository assets (including the project logo).
* corpus_preparation - Information about the corpus preparation strategy, and the corpus that was created for the study.
